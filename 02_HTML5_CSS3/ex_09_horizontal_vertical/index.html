Horizontal & Vertical Centering.

1. 수평(가로) 정렬

1.1 inline/inline-block

정렬 대상 요소(텍스트 또는 링크등의 inline 레벨 요소 또는 inline-block 레벨 요소)의 부모요소의 text-align:center;로 지정

.content{text-align:center;}

1.2 block

정렬 대상 요소에 너비를 명시적으로 지정하고 margin-right와 margin-left 프로퍼티에 auto 지정

정렬 대상 요소에 너비를 명시적으로 지정하지 않으면 너비가 full width가 되므로 중앙 정렬이 필요없음.

.item{width: 200px; height: 100px; margin: 10px auto; background: yellow;}

1.3 복수의 block

복수의 block 요소는 기본적으로 수직 정렬, 이것을 수평정렬하기 위해서는 정렬 대상 block요소를 inline-block요소로 변경한 후 부모 요소에 text-align: center;를 지정한다.

정렬 대상 요소에 width를 지정하지 않으면 콘텐츠에 너비에 맞춰 너비가 결정되므로 명시적으로 너비를 지정한다.

.content1{text-align: center;}
.item1{width: 200px; height: 100px; background: yellowgreen; margin-bottom: 1px; display: inline-block;}

1.4 Flexbox

Flexbox를 사용할 수도 있다. 정렬 대상 부모 요소에 선언을 해줘야 함.

.flex-center{ display: flex; justify-content: center; }

2. 수직(세로) 정렬

2.1 inline/inline-block 요소

2.1.1 single inline 요소

정렬 대상의 부모 요소에 padding-top과 padding-bottom 프로퍼티 값을 동일하게 적용한다.

.container{ border: 2px solid #000; padding: 20px; }

2.1.2 multiple line 요소

여러 줄의 텍스트의 경우 padding-top과 padding-bottom 프로퍼티 값을 동일하게 적용한다.

또 다른 방법으로 Vertical-align 프로퍼티를 사용하는 방법도 가능, 대신에 이 방법은 table 속성을 사용해야 한다.

.parent{ display: table; height: 100px; } .child{ display: table-cell; Vertical-align: middle; }

2.2 block 요소

2.2.1 요소의 높이가 고정되어 있는 경우

부모 요소를 기준으로 절대 위치를 지정한다.

.parent1{ position: relative; } .child1{ position: absolute; top: 50%; height: 100px; margin-top: -50px }

2.2.2 요소의 높이가 불확정 상태의 경우

부모 요소를 기준으로 절대 위치를 지정한다.

.parent2{ position: relative; border: 2px solid #000; height: 300px; } .child2{ width: 100px; background: rebeccapurple; position: absolute; top: 50%; transform: translateY(-50%) }

2.2.3 Flexbox

부모 요소에 Flexbox layout을 지정한다.

.parent3{ display: flex; flex-direction: column; justify-content: center; } .child3{ width: 100px; height: 50px; background: rebeccapurple; }

3. 수직, 수평(세로가로) 정렬

요소의 너비와 높이가 고정되어 있는 경우, 요소의 너비와 높이가 불확정 상태의 경우 모두 사용 가능한 방법

.parent4{ position: relative; } .child4{ position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%) }
