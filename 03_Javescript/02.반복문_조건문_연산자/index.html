자료형(기본 원시 타입 (Primitive)) -string, number, boolean, null, undefined, symbol

Number

  • 숫자형 - 실수형 정수형 구분하지 않고 8byte의 실수 형태로 처리
  • NaN(Not a number): 계산식의 결과가 숫자가 아님을 나타낸다.
  • infinity: 무한대를 나타내는 상수이며 어떤 수를 0으로 나누거나 infinity를 어떠한 수로 나눈 식의 결과를 나타냄

String

문자열 표현에 작은 따옴표, 큰따옴표 모두 사용 가능하지만 혼용해서 사용하지는 않는다.

boolean

비교 연산의 결과 값으로, true 또는 false값 중 하나를 가진다.

null은 값이 없거나 비어 있음을 뜻하고 undefined는 값이 초기화 되지 않았음(정의되지 않음)을 뜻함.

null과 undefined는 의미가 비슷하지만 값을 할당하지 않은 변수는 undefined를, 할당되고 코드에서 명시적으로 값이 없음을 나타낼 때는 null을 사용

객체타입(Object type) - function, arrays, classes
