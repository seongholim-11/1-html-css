함수 정의하기

함수를 실행하려면 함수를 정의하고 정의된 함수를 실행해야 하며, 함수는 불러줘야 실행한다.

        function 함수이름(){
            문장
        }

내장함수

  • perseInt() - 문자형 데이터를 정수형 데이터로 바꿈(소수점 없이)
  • perseFloat() - 문자형 데이터를 실수형 데이터로 바꿈(소수점 있이)
  • String() - 문자형 데이터로 바꿈
  • Number() - 숫자형 데이터로 바꿈
  • Boolean() - 논리형 데이터로 바꿈
  • IsNanN() - 'Not a Nuber' 숫자가 아닌 문자가 표현하면 true 값을 반환
  • eval("5+6+9+6") - 문자형 데이터를 큰따옴표가 없는 스크립트 코드로 처리하고 산술연산 해줌
  • alert() - 경고창
  • comfirm() - 확인(true) / 취소(false)
  • prompt - 입력 대화상자를 만들어 줌(묻고 답하기)

시간에 따른 메소드

  • 시간에 따라 명령을 수행하게 해주는 메소드

    setTimeout(a, b) - a: 함수/명령어, b: 지연시간(1/1000초)
    clearTimeout(a) - setTimeout() 설정한 내용을 중지 시킴

  • 시간에 따라 반복하여 명령을 수행하게 해주는 메소드

    변수 = setInterval(a, b, ) - a: 함수/명령어, b: 지연시간(1/1000초)
    clearInterval(a) - 변수에 정의된 setInterval() 의 실행을 중지 시킴
