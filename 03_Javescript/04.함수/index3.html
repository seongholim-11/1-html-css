이벤트 핸들러 종류

  • onBlur == 대상에서 포커스가 이동되었을 경우
  • onFocos == 대상에서 포커스가 들어왔을 경우
  • onClick == 대상에서 클릭했을 경우
  • onLoad == 대상에서 열었을 경우
  • onChange == 대상에 입력되어 있는 값이 바뀌었을 경우
  • onDbclick == 대상을 더블클릭했을 경우
  • onDragDrop == 대상이 클릭된 상태에서 이동했을 경우
  • onError == 대상에 에러가 발생했을 경우
  • onKeyDown == 키를 입력했을 경우
  • onKeyPress == 키를 누르고 있을 경우
  • onKeyUp == 키를 누르고 뗐을 경우
  • onMouseDown == 마우스를 눌렀을 경우
  • onMouseUP == 마우스를 뗐을 경우
  • onMouseOver == 마우스 포인터가 위치했을 경우
  • onMouseOut == 대상에서 마우스 포인터가 벗어났을 경우
  • onMouseMove == 마우스를 움직였을 경우
  • onRest == 대상을 재시작했을 경우
  • onMove == 윈도우나 프레임을 움직일 경우
  • onResize == 윈도우나 프레임의 크기를 변경했을 경우
  • onSelct == 입력 양식의 한 필드를 선택했을 경우
  • onSubmit == 입력 양식을 서버로 보냈을 경우
  • onUnload == 대상을 종료했을 경우
  • onAbort == 이미지를 읽다가 중지했을 경우

이벤트 객체 속성

  • clientX(or Y) == 현재 문서 기준으로 이벤트가 발생한 x(or Y)좌표(화면상의 높이, 너비)
  • pageX(or Y) == 현재 문서 기준으로 이벤트가 발생한 x(or Y)좌표(페이지 전체의 높이, 너비)
  • layerX(or Y) == position 속성값이 'absolute'인 레이어 안에서의 상대 X(or Y)좌표
  • screenX(or Y) == 모니터를 기준으로 이벤트가 발생한 x(or Y)좌표
  • taget == 이벤트가 발생한 대상 요소
  • button == 마우스 버튼(왼쪽, 오른쪽, 휠)의 클릭된 상태 체크
